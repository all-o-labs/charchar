01

The Problem

AI coding tools generate characters that all look the same — generic proportions, flat colors, no personality. You don't know the design vocabulary to ask for better.

Without charchar

  • Same body proportions every time
  • Random colors with no harmony
  • Unreadable silhouettes
  • Lifeless, static poses
  • Inconsistent style across characters

With charchar

  • Intentional proportions per character role
  • 60-30-10 color harmony
  • Clear, distinctive silhouettes
  • Personality-driven poses and expressions
  • Unified style across your entire cast
02

How It Works

1

Install

One command adds 19 character design skills to your project.

npx charchar
2

Auto-loaded

The core character-design skill loads automatically, injecting best practices into every character task.

3

Slash commands

Use /create-character, /pixel-art, /animate-char and more for specific tasks.

03

19 Skills

Create

/create-character

Full character from concept to code — silhouette, proportions, colors, details

/create-sprite

2D game sprites in pixel, vector, or canvas format with animation-ready poses

/create-3d

Three.js / WebGL characters with rigging, textures, and React Three Fiber support

/create-avatar

Profile avatars for GitHub, Slack, Discord — SVG-based, dark mode ready

Style

/pixel-art

Pixel art with palette limits (NES/SNES/GBA), dithering, and sub-pixel animation

/chibi

2-3 head ratio super-deformed style with exaggerated expressions

/realistic

Anatomically accurate 7.5-8 head ratio with lighting and material textures

/flat

Minimal flat design — no outlines, geometric shapes, 3-5 colors

/retro

8-bit / 16-bit era characters with hardware palette constraints and CRT effects

Enhance

/animate-char

Idle, walk, run, attack animations with CSS keyframes or Canvas frame loops

/expression-sheet

6 base expressions + extended set with intensity levels in a grid layout

/sprite-sheet

Pack frames into optimized sheets with TexturePacker-compatible JSON output

/color-palette

Generate harmonious palettes based on character personality and role

/turnaround

Front / three-quarter / side / back views with proportion consistency checks

Quality

/polish-char

Scan and improve: silhouette, color harmony, line quality, detail balance

/critique-char

Design critique with silhouette test, thumbnail test, and storytelling analysis

/harmonize

Unify style across multiple characters — line weight, palette, proportions

System

/teach-charchar

Scan your project's existing character assets and generate a .charchar config

04

Get Started

Claude Code

npx charchar

Skills are installed into your project's .claude/skills/ directory and loaded automatically.
